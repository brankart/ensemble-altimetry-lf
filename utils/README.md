# Utilities

Python tools that have been used to prepare data.
Not needed to reproduce the Mediterranean application,
since prepared data are available on a public repository.

* <i>prepare_obs_database.bash</i>: Prepare altimetric database from native along-track Aviso files.

* <i>extract_from_aviso_map.bash</i>: Extract Aviso mapped data from native file, and interpolate on model grid.

* <i>extract_ssh_lf.bash</i>: Extract low-frequency component from model data

* <i>extract_ssh_lf_recomb.bash</i>: Recombine domains (if parallelization)

* <i>extract_ssh_lf_aviso.bash</i>: Extract low-frequency component from aviso data

* <i>extract_ssh_lf_aviso_recomb.bash</i> : Recombine domains (if parallelization)
